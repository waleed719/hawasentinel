# AI Agents
5 agents active & working for you

### Sensor Agent
Monitoring real-time PM2.5, NO2, and SO2 data streams across 42 active regional grid sensors.

### Forecast Agent
Running predictive dispersion models for the next 48 hours based on current thermal inversion data.

### School Agent
Evaluating exposure risks for primary education facilities based on localized AQI spikes.

### Hospital Agent
Monitoring respiratory admission rates across 12 primary care centers in high-risk zones.

### Gig Worker Agent
Distributing N95 mask mandates and shift restriction advice to registered fleet platforms.

### Verifier Agent
Ensuring ground sensor anomalies align with historical patterns before escalating crisis level.
